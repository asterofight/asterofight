Astero Fight

Free MMO space shooter by many of us

Goal

Shoot everything that moves to win.

Spaceships

Fighter: all purpose spaceship

Controls

  • WASD and arrow keys: Accelerate
  • Left click: Use primary weapon
  • Right click: Use secondary weapon/equipment
  • Q: Mount/unmount big spaceships/stations

User Interface

  • Enter: Open chat window
  • /: Open chat window and can enter commands
  • T: Statistics
  • U: Switch spaceship (requires login)
  • N: Upgrades (requires login)

Chat Commands

  • /dbg: Debug
  • /swap: Join the other team
